# 指针相关
## 空指针
注意指针为空的情况，比如在迭代法二叉树的时候
```
class Solution {
public:
    vector<int> preorderTraversal(TreeNode* root) {
        stack<TreeNode*> st;
        vector<int> result;
        if(root==nullptr) return result;
        st.push(root);
        while(!st.empty()){
            root=st.top();
            result.push_back(root->val);
            st.pop();
            if(root->right!=nullptr) st.push(root->right);
            if(root->left!=nullptr) st.push(root->left);
        }
        return result;
    }
};
```
中`if(root==nullptr) return result;`要进行判断，因为开始root有空的情况
